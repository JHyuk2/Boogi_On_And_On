# 🌊 부기온앤온 (Boogi On & On) 코어 기획서 v1.1
Core Slogan: "가장 작고 다정한 첫걸음"

1. 존재 이유 (The Why) : 우리는 왜 이 앱을 만드는가?
우리는 더 효율적인 기계가 되라고 채찍질하는 세상의 공식에 반기를 듭니다. '자기 자비(Self-Compassion)'라는 단 하나의 철학을 관통하여, 상처받고 멈춰 선 사람들을 위한 가장 안전한 바다를 만듭니다.

🎯 [핵심 타겟과 문제 인식] : 누구를, 무엇으로부터 구하는가?
우리가 구하려는 사람들: 최선을 다하지 않은 게 아닌데 결과가 따라주지 않아 자존감이 바닥난 사람들. 쳇바퀴 같은 일상 속에서 자기 비하에 빠져 '나 스스로 괜찮다고 말해주는 것조차 가식으로 느껴지는' 번아웃 상태의 유저들.

우리가 부수려는 기존의 공식: 달성하지 못한 목표를 빨간색으로 칠해 죄책감을 주고, 하루를 빼먹으면 연속 기록(Streak)을 깨버려 유저를 탈락시키는 '독성 긍정(Toxic Positivity)' 기반의 갓생(God-생) 투두 앱과 숨 막히는 시간 관리 커리큘럼.

🗺️ [세계관 정의] : 여행자와 바다
우리는 유저를 '오늘 할 일을 쳐내야 하는 노동자'로 취급하지 않습니다. 유저는 인생이라는 거대한 바다를 각자의 속도대로 탐험하는 '여행자'입니다. 빠르게 헤엄쳐야 할 의무는 없습니다. 폭풍우가 치는 날에는 파도에 몸을 맡기고 가만히 둥둥 떠 있는 것(숨만 쉬는 것)조차 훌륭한 항해의 일부로 인정하는 세계관입니다.

💎 [4대 존재 가치] : 이 앱이 여행자에게 주는 4가지
위로 (Comfort) : "네가 존재해 줬으면 해" 기쁨을 강요하지 않고 유저의 슬픔을 있는 그대로 수용하는 위로입니다. *"아무것도 하지 못했지만, 네가 웃지 못하는 날에도 살아가 줬으면 좋겠다"*는 조건 없는 존재의 승인을 제공합니다.

추진력 (Momentum) : "가장 작고 다정한 첫걸음" 의지가 꺾인 사람에게 거대한 목표는 쳐다보기조차 싫은 거대한 벽입니다. 우리는 이 벽을 조약돌처럼 잘게, 뇌가 저항을 포기할 정도로 하찮게 부숴줍니다. 마찰력 제로의 아주 작은 물리적 행동(징검다리)에서 추진력이 시작됨을 증명합니다.

자기 효능감 (Self-Efficacy) : "내 속도대로 잘 가고 있어" 남들과 비교하는 속도가 아니라, 어제보다 오늘 조금 더 나아갔다는 나만의 흔적을 시각화합니다.

내일에 대한 기대 (Hope for Tomorrow) : "무사히 버틴 오늘" 유저가 잠들기 전 앱을 끄며 *"오늘 하루도 꽤 잘 버텨냈어. 내일의 항해도 나쁘지 않을 거야"*라고 생각하게 만들어, 내일을 두려워하지 않게 합니다.

2. 무엇을 어떻게(How) 해결하고 싶은가?
2-a. 위로 (Comfort) : 슬픔을 수용하는 덤덤한 다정함
기계적인 푸시 알림과 독성 긍정을 철저히 배제합니다. 유저의 '슬픔'과 '무기력'을 있는 그대로 수용하고 곁에 머물러 줍니다.

🐢 전달자의 페르소나 (겉바속촉 부기): "넌 할 수 있어!"라며 방방 뛰지 않습니다. 인생이 내 맘 같지 않다는 걸 이미 아는 느긋하고 덤덤한 오랜 친구입니다.

📦 아침의 체크인 (슬픔을 허락하는 UI): [☀️ 맑음], [☁️ 무난], [🌧️ 흐림] 버튼. 특히 '흐림'을 선택하면 목표를 강요하지 않고, [오늘은 그냥 쉴래] 버튼을 통해 유저의 온전한 '쉼'을 공식적으로 승인합니다.

💌 예상치 못한 위로: 유저가 가장 고립감을 느낄 타이밍(새벽 접속 등)에 할 일을 재촉하는 대신 "오늘 하루도 잘 보냈길 바라" 같은 조건 없는 다정함을 먼저 보냅니다.

2-b. 추진력 (Momentum) : 3단계 항해 내비게이션과 생태계
아무리 따뜻한 위로를 받아도 눈앞의 목표가 거대하면 숨이 막힙니다. 부기온앤온은 인지적 부하(생각)와 물리적 행동을 완벽하게 분리하여 시작의 허들을 없앱니다.

🗺️ ① 3단계 징검다리 시스템 (거시적 조망과 미시적 실행의 분리):

1단계 (4개의 큰 대륙): 추상적인 목표를 입력하면, AI가 시간순으로 4개의 거대한 마일스톤(섬)을 세워 전체 지도를 보여줍니다.

2단계 (작은 섬들의 군도): 큰 섬이 부담스러울 때, AI가 이를 완료 가능한 단위의 하위 프로젝트(명사형 개념)로 해체합니다.

3단계 (퐁당퐁당 징검다리): 뇌가 저항할 때 [🪨 징검다리 놓기] 버튼을 누릅니다. AI는 인지적 고민을 배제하고 무조건 10분 이내에 끝나는 '물리적 행동(예: 책상에 앉아 펜 꺼내기)'으로만 3개의 징검다리를 생성합니다.

⛵ ② 항해 게시판 (템플릿 갤러리 - 진입장벽 제로화): * "목표를 적어보세요"라는 빈 도화지의 공포를 없앱니다. 다른 유저들이 이미 쪼개놓은 훌륭한 항로를 구경하고, 버튼 한 번에 내 바다로 복사해 올 수 있습니다.

예시: [막막한 신혼집 구하기 첫걸음 (예산부터 부동산 방문까지)], [야근 없는 법인세 신고 시즌, 결산 스케줄 쪼개기]

🔋 ③ 행동의 에너지 변환 (즉각적 보상): 쪼개진 하찮은 징검다리를 밟을 때마다 에너지 구슬이 부기에게 흡수되며, 부기가 다음 섬을 향해 헤엄쳐 나아가는 동력이 됩니다.

🪸 ④ 죄책감 없는 바다 잔디 (해빗 트래커): 연속 기록(Streak)이 끊겼을 때의 스트레스를 없앱니다. 아무것도 못 한 날은 실패(X)가 아니라, 부드러운 '물결표(~)'로 채워지며 *"파도에 몸을 맡기고 흘러간 날"*로 기록됩니다.

2-c. 자기 효능감 (Self-Efficacy) : 존재를 긍정하는 안전망
억지 긍정이 통하지 않는 번아웃의 임계점에서 유저를 보호합니다.

🫁 상시 방어선 (숨 쉬기): 앱 최상단에 고정된 삭제 불가능한 퀘스트. 모든 걸 실패해도 이 버튼 하나면 *"살아있어 줘서 고마워!"*라며 하루를 긍정해 줍니다.

🔥 최후의 은신처 (SOS 모닥불): 스스로를 비하하게 되는 밤, 모닥불 아이콘을 누르면 모든 생산성 UI가 물속에 잠깁니다. 오직 타닥타닥 타는 장작 소리와 함께, 내가 썼던 다짐들이 영화 엔딩 크레딧처럼 올라오는 디지털 응급처치실입니다.

2-d. 익명의 연대 (Anonymous Solidarity) : 은둔 고수들의 바다
날 선 비판은 도려내고 솔직함만 남긴 청정 구역입니다.

💌 유리병 편지: 누구에게도 말 못한 외로움을 적어 밤바다에 띄웁니다.

🤝 은둔 고수들의 메아리: 나와 비슷한 아픔을 겪은 타인들의 현실적인 답장이 돌아옵니다. 누군가를 위로하며 스스로 치유되는 선순환 구조.

🛡️ 저비용 AI 문지기: LLM을 통해 공격성 수치가 높은 편지는 바다에 띄워지기 전 사전에 부서지게 필터링합니다.

3. 🗺️ [부기온앤온] 전체 화면 및 기능 구조도 (IA)
🚪 3-1. 첫 화면 : 나의 바다 (항해의 시작과 공유)
유저가 앱을 켜면 가장 먼저 마주하는 활기차고 다정한 항해 준비 공간입니다.

[상단] 거대한 목표 생성기: 바다 위에 떠 있는 [ + 나만의 목표 생성 ] 큼직한 버튼. 터치 시 AI 부기와의 3단계 징검다리 쪼개기가 시작됩니다. (발행 시 기본값은 '공개'로 설정되어 생태계를 구축하되, 원하면 비공개 처리 가능)

[중앙~하단] 항해 게시판 (Template Gallery): 인기/최신/테마별 카테고리 탭. 핀터레스트처럼 카드형으로 정렬된 다른 여행자들의 항로를 탐색하고 [내 바다로 가져오기]를 할 수 있습니다.

🏝️ 3-2. 항해 지도 (실행과 시각화)
차가운 텍스트 리스트를 완전히 버린, 나만의 속도로 유영하는 인터랙티브 지도입니다.

여정 지도 모드 (Milestone Map): 세로형 스크롤 보물지도. 큰 섬 4개가 보이고, 그 사이를 잇는 징검다리들이 놓여 있습니다.

징검다리 놓기 UX: 다음 섬이 너무 멀어 보일 때 깃발 아래의 [🪨 징검다리 놓기]를 누르면, 물보라와 함께 10분 컷 물리적 미션 3개가 퐁당퐁당 솟아오릅니다.

바다 잔디 모드 (Ocean Heatmap): 토글 전환 시 나타나는 이번 달의 기록. 산호초(성공)와 물결표(휴식)가 빼곡히 채워진 바다 밑바닥 달력을 봅니다.

🏖️ 3-3. 나의 해변 (위로와 익명 연대)
실행의 압박에서 벗어나 철저히 정서적 충전을 하는 공간입니다.

유리병 편지함: 파도를 타고 밀려온 다른 사람들의 익명 고민을 읽고 다정한 답장을 보냅니다. 스마트폰을 위로 스와이프하여 내 고민을 던질 수도 있습니다.

오늘의 감정 체크인: 맑음/무난/흐림 박스를 터치해 부기와 조용히 교감합니다.

🎒 3-4. 여행자 가방 (기록과 설정)
여행자의 여권: 가입 시 적었던 '나를 위한 서약서'가 각인된 프로필.

셋 로그 (Set Log): 징검다리를 밟고 남긴 사진들이 폴라로이드처럼 걸려 있는 갤러리.

부기의 상점: 획득한 진주로 부기의 소소한 꾸미기 아이템(밀짚모자 등) 교환.

4. 수익성 모델 (Monetization) : 힐링을 해치지 않는 과금
노골적인 배너 광고나 강제 팝업은 멘탈케어 앱에 독약입니다.

☕ 자발적 후원 ("부기 밥값 보태기"): 소액 후원 시 부기의 비밀 감사 편지와 황금 조개 스탬프 제공.

🎒 수집/여행 요소: 위트 있는 부기 코스튬 및 프리미엄 밤바다 테마 소액 결제 판매.

🕊️ 착한 광고 (선택형): 화면 구석에 날아오는 '갈매기'를 터치할 때만 보상형 광고 시청. 소액 결제로 [갈매기 출입 금지 표지판] 영구 소장권 판매.

5. 기술 사용 명세서 (Tech Architecture & Cost)
서버비를 극한으로 방어하는 온디맨드(On-Demand) AI 최적화 아키텍처.

📱 프론트엔드 : Flutter + Rive/Lottie (iOS/Android 동시 배포 및 가벼운 인터랙션 애니메이션)

⚙️ AI 에이전트 : Antigravity 2.0 (코드 작성 및 프로젝트 관리 지휘관)

🧠 기능형 AI API : OpenRouter 연동

유저가 [징검다리 놓기]를 누를 때만 API를 호출하는 점진적 생성(Lazy Loading)으로 토큰 소모 최소화.

공용 DB 캐싱을 통해 대중적인 템플릿 호출 시 서버비 0원 구현.

☁️ 백엔드/DB : Supabase (또는 Self-hosted PostgreSQL) (Firebase 대안, SQL 기반 안전한 데이터 관리)

🔄 CI/CD : GitHub Actions + Fastlane (무중단 백그라운드 배포 자동화)
