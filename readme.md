# 🐢 Boogi On & On (부기온앤온)

> *"완벽하지 않아도 괜찮아, 오늘의 하찮은 물장구가 너를 바다로 이끌 테니까."*

**Boogi On & On**은 완벽주의와 번아웃에 지친 현대인을 위한 **'자기 자비(Self-Compassion)' 기반의 AI 마이크로 스텝 투두(To-Do) 어플리케이션**입니다. 달성하지 못한 목표에 빨간 줄을 그으며 죄책감을 주는 대신, 아무리 거대한 목표라도 당장 오늘 할 수 있는 가장 하찮고 귀여운 4개의 '물장구(행동)'로 쪼개어 다정한 위로와 작은 성취감을 제공합니다.

## ✨ Key Features (주요 기능)

### 1. Two-Track UI (투 트랙 목표 설정)
유저의 당일 컨디션과 의도에 맞춰 두 가지 목표 설정 모드를 제공합니다.
* 💧 **[작은 물장구 치기] (Bottom-up):** 당장 오늘 해치워야 할 일상적인 단기 할 일(예: 청소하기, 찌개 끓이기)을 입력하면, 즉시 '오늘의 하찮은 4단계 행동'을 제공하여 일상적인 리텐션과 도파민을 부여합니다.
* ⛵ **[먼 바다로 항해하기] (Top-down):** 장기 프로젝트(예: 이직, 신혼집 마련)를 입력하면, AI가 거대한 목표를 4개의 '큰 섬(마일스톤)'으로 쪼개어 시각적인 로드맵을 그려줍니다.

### 2. 2-Depth 직통 구조 (인지적 과부하 방지)
* 거시적 목표의 '큰 섬(마일스톤)'을 클릭하면, 복잡한 하위 카테고리로 계속 들어가는 대신 **곧바로 해당 섬에 닿기 위한 '오늘의 물장구 4개'**를 즉시 출력합니다. 유저의 심리적 압박감과 클릭 피로도를 최소화합니다.

### 3. Tagging & Visual Reward (모래 징검다리 시스템)
* 단기 할 일(물장구)을 장기 목표(먼 바다)에 태깅(Tagging)하여 연결할 수 있습니다. 
* 목표가 연결된 물장구를 완료하면, 바다 아래에서 픽셀 모래 뭉치가 솟아오르며 다음 섬으로 향하는 **'모래 징검다리'**가 시각적으로 완성됩니다. 소소한 일상이 거대한 목표로 연결되는 벅찬 성취감을 제공합니다.

### 4. Guilt-Free Widget (다정한 위젯)
* 앱에 며칠간 접속하지 않아도 '밀린 할 일'을 띄워 유저를 압박하지 않습니다. 
* 미접속 일수에 따라 마스코트 '부기'가 등껍질 속에서 쿨쿨 잠을 자거나, 등껍질 위에 모래가 쌓이며 평화롭게 기다리는 동적 위젯 연출을 통해 유저의 죄책감을 덜어줍니다.

## 🛠 Tech Stack (기술 스택 및 환경)

### Client & Engine
* **Godot 4 (GDScript):** 2D 픽셀 아트 애니메이션 및 파티클 연출에 최적화된 게임 엔진. 파이썬(Python)과 유사한 문법 구조를 가진 GDScript를 활용하여 1인 개발 속도를 극대화합니다.
* **Optimization (최적화):** 일반 앱 환경에서의 배터리 소모 및 발열 방지를 위해 `OS.low_processor_usage_mode = true` (저전력 모드) 및 `Compatibility` (호환성) 모드를 렌더러로 채택하여 모바일 네이티브 환경(iOS/Android)에 최적화했습니다.

### AI & API
* **Google Gemini API:** 유저의 프롬프트를 분석하고, 마일스톤을 4단계로 분할하여 JSON 형태로 파싱/응답하는 핵심 대화 로직을 담당합니다.

### AI Agents & Tools (1인 개발 툴셋)
* **Cursor IDE:** GDScript 및 API 통신 관련 코드 자동화 작성 어시스턴트.
* **Leonardo.ai:** 16-bit 픽셀 아트 스타일의 일관성 있는 게임 에셋(부기, 바다, 모래 등) 생성.
* **v0.dev:** 미니멀하고 투명한 바다 테마의 UI/UX 프로토타입 레이아웃 설계.

  - 빈 텍스트창 입력값을 AI로 전송 및 4단계 분할 텍스트 화면 출력 성공.

- [ ] **Phase 3: 그래픽 에셋 및 시각적 보상 적용**
  - Leonardo.ai 기반 2D 픽셀 캐릭터 및 배경 삽입.
  - 태깅 시스템 연동 및 '모래 징검다리' 파티클/애니메이션 구현.
